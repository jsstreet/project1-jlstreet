Rasberry Pi Hardware Extensions

One of my first projects focused on implementing hardware-based systems using a Raspberry Pi. I worked with a variety of attachments and peripherals, including stepper motors, motion sensors, keypads, and LED displays, to really understand how software controls hardware and how to better manipulate circuits. This project emphasized wiring, signal handling, and writing code that interacts cleanly with physical components. Working with some many different extensions of the same basic circuit improved my understanding of how wires and a CPU interact with devices. Sorting through hardware constraints and debugging real-world behavior has helped strengthen my understanding of how software and circuits come together in practical systems.
